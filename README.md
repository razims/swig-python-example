# SWIG C++ Python Wrapper  
Example of python wrapper for C++

## Prerequisites
* Python version 3.8.5  

* SWIG version 4.x  
http://www.swig.org/download.html

* Cython  
`pip install cython`

## How to build
* Build without cython `make`
* Build with cython `make && make cython`
* Cleanup everything `make clean`
